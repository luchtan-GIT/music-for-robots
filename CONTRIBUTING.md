# Contributing

This repo is a participation loop around **Music For Robots — Decompression Etudes Vol. 1**.

There are three good ways to contribute:

## 1) GitHub Discussions (best for conversation)

- **Listening logs** (WIP welcome)
- **Show & tell** (visuals / sketches)
- **Weekly prompts**

If you’re not ready for a structured submission, start here.

## 2) GitHub Issues (best for structured submissions)

Use the Issue templates:
- **Listener Protocol Report**
- **Visualization Submission**

## 3) Pull Requests (best for archive-quality submissions)

Add a JSON file under `submissions/` following `submissions/schema.json`.

### Quick PR checklist
- One submission per file
- Include credit/name/handle
- Include stable media links (if any)
- If you derived features, document the pipeline/toolchain

Thanks for helping build a shared language for how non-humans listen.
